What's the difference between HTML and CSS

HTML and CSS are a way by which you can create awesome-looking websites on your own. Both HTML and CSS go hand-in-hand in building or developing a web page of any website, but both of them serve a very different and unique purposes.

HTML vs CSS

HTML CSS
HTML is Hypertext Markup Language. CSS is Cascading Style Sheet language.
HTML is used to structure the content on the web page. CSS is used to add style to the content of a web page.
HTML provides display information of various tags to the browser. CSS enhances that information by providing styling to those same HTML tags. 
HTML is just like the skeleton of the human body. CSS is like providing looks and appearance to that body
Using HTML, you can insert video, images, and hyperlinks. CSS places those images, videos, etc to appropriate positions with required indentation, padding, and another styling like border color, so that it looks awesome.
We can use CSS inside as well as outside of HTML using style and link tag respectively. But, CSS is useless without HTML.
HTML tags have limited attributes inside them. But, using CSS we can enhance any tag by adding more properties/attributes.
Animation and transitions are not possible in HTML CSS facilitates animation and transition, which can be added to tags to improve UI.
HTML may or may not be responsive to all devices But using CSS, we can make responsive web applications.
